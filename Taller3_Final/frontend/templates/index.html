🏗️ Arquitectura de Microservicios

Este sistema está construido con una arquitectura de microservicios que incluye: un API Gateway como punto de entrada central, tres microservicios independientes (Usuarios, Proyectos, Tareas) con FastAPI y PostgreSQL, y un Frontend con Flask. Todo orquestado con Docker Compose.
